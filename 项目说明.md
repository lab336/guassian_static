# guassian_static —— 多相机静态/动态采集到 3DGS 的处理流水线

围绕**固定多相机阵列**采集的素材，完成「视频切帧 → 抠像 → 标定参数转换 → 去畸变 → 点云重建」全过程，
最终产出可用于 **3D Gaussian Splatting (3DGS)** 训练的数据，或独立的三维点云。

本仓库把不同环节拆成若干模块，部分模块直接复用了第三方开源项目（以 git submodule 形式引入）。

---

## 总体数据流

```
采集端工具(Zcam/)            多相机视频/照片 + 背景板
        │
        ▼
preprocess/batch_cut_video.py     视频 → 按帧分组的图像 (by_frame / by_video)
        │
        ├────────────► background/   抠像，得到前景 fgr / alpha pha（可选）
        │
        ▼
Convert/                          标定参数转换 + 去畸变 + 主点居中 → COLMAP/PINHOLE
        │
        ├────────────► 3DGS 训练（外部）
        │
        ├──► colmap_pipeline.py    固定阵列 COLMAP 多视图 MVS，逐帧出点云
        ├──► points/               RoMa v2 稠密三角化重建点云 + 裁剪/可视化
        └──► SFS/                  基于掩码的轮廓重建 (Shape from Silhouette)
```

各模块可独立运行，按需组合。

## 模块说明

### `Convert/` —— 标定转换与去畸变（核心）
把 calib.io/libCalib 的 `calib.json` 或 Agisoft 的 `cameras.xml`，转换成 COLMAP 参数并对图像去畸变、裁黑边、主点居中，
产出可直接用于 3DGS 训练的 `images/ + sparse/0/`。
**每个脚本的功能、数据准备、命令、产出见 [`Convert/README.md`](Convert/README.md)。**

### `preprocess/` —— 视频批量切帧
`batch_cut_video.py`：把一个目录下的多段视频（多相机）批量抽帧，多进程加速。支持按 `target_fps` 降采样、`num_frames` 限制总帧数、`uniform/head` 采样、`seek` 快速跳帧。
- 数据准备：`videos/` 下若干 `.mp4`。
- 命令：`python preprocess/batch_cut_video.py --data_dir ./videos --output_dir ./frames --target_fps 30 --output_mode both`
- 产出：`by_video/<视频名>/1.jpg...`（同一视频帧聚一起）和 `by_frame/<帧号>/1.jpg...`（同一帧的各相机聚一起，正是 `Convert/` 的输入）。

### `background/` —— 前景抠像（第三方：BackgroundMattingV2）
基于论文 *Real-Time High-Resolution Background Matting* 的现成模型。需要**额外拍一张干净背景板**，即可对采集图像抠出前景 `fgr` 和透明度 `pha`（alpha）。这里作为整体工具使用，不逐文件解释。
- 数据准备：前景图目录 + 对应背景板目录；权重 `background/checkpoint/pytorch_resnet101.pth`。
- 命令：`python background/inference_images.py --images-src <前景目录> --images-bgr <背景目录> --output-dir output --output-types com pha fgr`
- 产出：`com`（合成）/`pha`（alpha 掩码）/`fgr`（前景）。其中 `pha` 可交给 `SFS/pha_to_mask.py` 转二值掩码做轮廓重建。

### `Zcam/` —— 采集端辅助小工具
拍摄阶段使用的桌面/整理工具，整体说明：
- `F/`：**调焦助手** GUI（`main.py`，ttkbootstrap + mss 屏幕取流），帮助多 ZCAM 相机对焦；`build.py`/`build_config.py` 用 PyInstaller 打包成 exe。
- `framecat/`：**视频同帧对比工具** GUI（`gui.py` + `frame_core.py`），把多段视频抽同一帧拼图对比。
- `file/reorganize_cameras.py`：把采集落盘的 `data/<时间戳>/xxx.png` 重组为 `reorganized/Camera_XX/1.png...`。

### `colmap_pipeline.py` —— 固定阵列 COLMAP 重建
针对「相机位置固定、场景随帧变化」的数据，封装 COLMAP 命令行做标定 + 逐帧 MVS，输出稀疏参数与逐帧稠密点云。需安装 COLMAP。
- 数据准备：`dataset/frame_000/{0.png,1.png,...}`、`frame_001/...`。
- 命令：`python colmap_pipeline.py --dataset ./dataset --output ./output --calib_frame 0`
  （`--skip_mvs` 只标定 / `--skip_calibration` 只 MVS / `--frame_range 0:9` 指定帧段 / `--no_gpu`）
- 产出：`output/sparse/0/`（参数）、`undistorter_images/<帧>/`、`points_cloud/<帧>.ply`。

### `points/` —— 稠密三角化重建与点云工具
- `triangularization.py` / `triangularization_strict.py`：从去畸变多视图图像 + 已知内外参，用 **RoMa v2** 做稠密匹配三角化生成点云（`_strict` 为更严格的多视图一致性/置信度过滤版）。需 `romav2`、torch。
  命令示例：`python points/triangularization_strict.py --data_dir ./data2/dense --output_dir ./data2/dense/points --num_samples 8000 --neighbor_range 5`
- `crop_ply_batch.py`：按 `crop_range.txt` 里的 X/Y/Z 范围批量裁剪 PLY。`python points/crop_ply_batch.py crop_range.txt <input_folder> [output_folder] [--invert]`
- `ply_viewer.py`：交互式查看 PLY，拖动滑块实时裁剪，导出范围到 `crop_range.txt`（与上面的批量裁剪配套）。`python points/ply_viewer.py file.ply`

### `SFS/` —— 轮廓重建（Shape from Silhouette）
用各视角掩码体素雕刻出三维形状，作为一条不依赖纹理匹配的重建路线，整体说明：
- `pha_to_mask.py`：把抠像得到的 `pha`（alpha）阈值化成黑白二值掩码。
- `SFS.py`：读 `calib.json` + 各视角二值掩码，体素雕刻生成 `reconstruction.ply`（路径写在脚本顶部）。
- `del_points.py`：去掉点云内部点只保留表面。
- `project_pointcloud.py`：把点云投影回各相机视角，验证重建质量。
- `calib_to_colmap.py`：与 `Convert/` 同名脚本一致，供 SFS 流程内单独使用。

### `colmap/` —— 暂不展开
内含 `superglue_colmap.py` 及第三方 submodule **SuperGluePretrainedNetwork**、**WAFT**（光流），用于基于特征匹配/光流的 COLMAP 重建。**按要求本说明暂不解释。**

## 目录速览

| 目录/文件 | 作用 | 是否需第三方/权重 |
|---|---|---|
| `Convert/` | 标定转换 + 去畸变（核心，详见其 README） | OpenCV / 单目录模式需 pycolmap |
| `preprocess/` | 视频批量切帧 | OpenCV |
| `background/` | 前景抠像（BackgroundMattingV2） | 需权重 `checkpoint/*.pth` |
| `Zcam/` | 采集端 GUI/整理工具 | tkinter/ttkbootstrap |
| `colmap_pipeline.py` | 固定阵列 COLMAP MVS | 需安装 COLMAP |
| `points/` | RoMa v2 稠密重建 + PLY 裁剪/查看 | 需 romav2 / torch |
| `SFS/` | 轮廓体素雕刻重建 | OpenCV |
| `colmap/` | SuperGlue/WAFT 重建（暂不展开） | git submodule |
